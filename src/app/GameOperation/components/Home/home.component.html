Secret Santa

Please follow below steps to play a game.

Step 1 Sign In or Sign Up
  • To start with the game, you need to sign up with us.
  • If you already have a account, please Sign In.
Step 2 Group Creation
  • After successful sign-in or sign-up, please create group and add group members.
  • Add your name as member too if you want to play game.
Step 3 Group Set Up
  • Please provide Date of the gift exchange.
  • Now, we are all set to start with game.
Step 4 Password Set Up
  • All members will receive a mail including "Group Code" and password setup link.
  • Please set up password & sign-in to draw names.
Step 5 Draw Names
  • All members should draw name to start a game.
  • Every member will have one child.
Step 6 WishList
  • All members can add their Wishlist.
  • This Wishlist will be visible to respective Santa.
Step 7 Gift Exchange
  • On date of gift exchange, admin can view all Santa & child names.
Step 8 Enjoy
  • Keep your task ready & ENJOY...!!
